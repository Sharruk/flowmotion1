FlowMotion is a Django-based full-stack web application running locally on Ubuntu Linux at:

http://127.0.0.1:8000


This project does NOT use browser notifications.

All reminders must be sent using Linux system notifications via:

notify-send


The website itself does NOT push notifications.
Instead, Django runs a background scheduler that checks time and tells Linux to show notifications.
